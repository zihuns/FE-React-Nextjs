# FE-React-Nextjs

React 및 Next.js 기반의 프론트엔드 프로젝트와 학습 코드를 모아둔 저장소입니다.

## 🛠️ 기술 스택
- **Core:** React, Next.js
- **Language:** JavaScript, TypeScript
- **Package Manager:** npm

## 포함된 프로젝트

### 🛍️ react-shopping-mall
- React 기반의 쇼핑몰 애플리케이션
- 상품 리스트, 장바구니, 상세 페이지 구현
- **주요 라이브러리:** React Router, Styled-components

### 📚 react-bookshelf
- 도서 관리 및 리뷰 기록 애플리케이션
- 책 검색 및 나만의 서재 꾸미기 기능
- **주요 라이브러리:** React Router, Context API

### 📖 do-it-react
- 'Do it! 리액트' 교재 학습 예제
- 컴포넌트, 상태 관리 등 기초 실습 코드
- **주요 라이브러리:** Redux, react-redux, node-sass, json-server

### 🔥 nextjs-warmup
- Next.js 기초 학습 및 워밍업
- 프로젝트 초기 설정 및 라우팅 연습
- **주요 라이브러리:** Next.js

### 🛒 inflearn-shoppingmall
- 인프런 강의 기반 쇼핑몰 프로젝트
- 상품 리스트 및 장바구니 기능 구현
- **주요 라이브러리:** React, React Router

### ⚛️ react-warmup
- React 기초 학습 및 워밍업
- 컴포넌트 구조 및 Hooks 활용 연습
- **주요 라이브러리:** React Hooks

## 🚀 실행 방법

각 프로젝트 디렉토리로 이동하여 다음 명령어를 실행하세요.

```bash
cd [프로젝트명]
npm install
npm start
```
